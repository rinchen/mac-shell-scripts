# mac-shell-scripts
Joey's personal Mac shell scripts

This assumes that you have installed brew.sh.

## Files

| File | Description |
|------|-------------|
| `config.fish` | Fish shell config — sets PATH, env vars, and aliases (bat, neovim, atuin, iTerm2 integration) |
| `installbrewitems.sh` | Installs all Homebrew packages and casks for a fresh Mac setup |
| `tmux.conf` | tmux config — keybindings, status bar, mouse support |
| `update.sh` | Runs `topgrade` to update all system packages |
